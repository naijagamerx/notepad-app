Rich Text Editing

  • Lists and checkboxes

Note Management

  • Create unlimited folders
  • Drag and drop notes
  • Export folder contents
  • Bulk operations

Smart Search

  • Real-time search results
  • Search in titles and content
  • Advanced filtering options
