NotePad Documentation

A beautiful and functional web-based note-taking application

Introduction

NotePad is a modern, responsive web application that provides a clean and intuitive note-taking experience. Built with pure JavaScript, HTML5, and CSS3, it offers a seamless experience across all devices.

Installation

The application is built with vanilla JavaScript and doesn't require any build tools or dependencies. Simply follow these steps:

  1. Extract the downloaded zip file
  2. Open index.html in your web browser
  3. Start creating notes!

Customization

Colors

The main colors can be customized in the styles.css file:

/* Main colors */
.main-content {
    background-color: #fffef0; /* Background color */
}

.main-content::before {
    background-image: linear-gradient(#e5e5dd 1px, transparent 1px); /* Grid lines */
}

Layout

The sidebar width and main content layout can be adjusted in styles.css:

.sidebar {
    width: 300px; /* Sidebar width */
}

.note-editor {
    padding: 20px 40px; /* Content padding */
}
